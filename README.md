# Sacred Words Daily

A poetry and scripture community web app built with Vue 3, TypeScript, and Vite.

## Pages

- `/` — Home
- `/poetry` — Poetry
- `/quotes` — Inspirational Quotes
- `/bible-verses` — Bible Verses

## Project Setup

```sh
npm install
```

### Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```

### Preview Production Build

```sh
npm run preview
```

## Tech Stack

- [Vue 3](https://vuejs.org/) with `<script setup>` + TypeScript
- [Vue Router](https://router.vuejs.org/)
- [Vite](https://vite.dev/)
